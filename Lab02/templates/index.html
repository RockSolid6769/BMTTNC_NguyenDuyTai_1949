About This Lab


Advanced Information Security Practice

This web application demonstrates several classical cryptography algorithms used to encrypt and decrypt text messages.

The system currently supports:

• Caesar Cipher • Vigenere Cipher • Playfair Cipher • Rail Fence Cipher • Transposition Cipher

Available Algorithms


Caesar Cipher

Shifts letters of the alphabet by a fixed key.


Vigenere Cipher

Uses a keyword to perform polyalphabetic substitution.


Playfair Cipher

Encrypts pairs of letters using a 5x5 matrix.


Rail Fence Cipher

Encrypts text using a zig-zag pattern.


Transposition Cipher

Rearranges character positions using a key.
